# GitHub Issue Templates

## Purpose
The Open EU AI Operator uses GitHub issue templates to collect structured proposals from contributors.

## Added Templates
```text
.github/ISSUE_TEMPLATE/open-eu-agent-template.md
.github/ISSUE_TEMPLATE/open-eu-workflow-template.md
```

## Agent Proposal Template
Use this when a contributor wants to propose a new AI agent.

Required fields:
- Agent Name
- Sector
- Purpose
- Intended Users
- Inputs
- Outputs
- Human Review Gate
- Privacy / Compliance Notes
- Example Prompt
- Success Criteria

## Workflow Proposal Template
Use this when a contributor wants to propose a practical SME workflow.

Required fields:
- Workflow Name
- Sector
- Business Goal
- Required Inputs
- Steps
- Outputs
- Human Review Point
- Automation Ideas
- Demo Data
- Success Criteria

## Review Rule
Every issue should be reviewed for:
- SME usefulness
- Privacy risk
- Human review gate
- Compliance claims
- Reusability
