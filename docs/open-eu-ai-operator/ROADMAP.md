# Open EU AI Operator — Roadmap

## North Star
Create a public, open-source AI operator layer for European SMEs that helps businesses adopt AI with practical workflows, privacy awareness, and sector-specific templates.

## Phase 0 — Foundation
Status: In Progress

Deliverables:
- README
- Agent Registry
- Notion Command Center
- Task Tracker
- Roadmap
- Compliance checklist stubs

Success criteria:
- GitHub module exists
- Notion tracker exists
- v0.1 scope is clear

## Phase 1 — v0.1 Public Pack
Target deliverables:
- Hospitality OS starter workflow
- Real Estate OS starter workflow
- Marketing OS starter workflow
- GDPR checklist
- EU AI Act checklist
- Contributing guide
- Demo prompts

Success criteria:
- A small business can copy the templates and run its first AI workflow in one day.

## Phase 2 — Data Layer
Target deliverables:
- Supabase schema
- Organization / location / user tables
- Lead and client tables
- Agent run logs
- Audit logs

Success criteria:
- Operators can track AI runs without exposing sensitive data.

## Phase 3 — Web Dashboard
Target deliverables:
- Public landing page
- Operator dashboard
- Agent cards
- Workflow cards
- Compliance status cards

Success criteria:
- The project can be demonstrated to SMEs, partners, and contributors.

## Phase 4 — Community Launch
Target deliverables:
- GitHub Issues templates
- Contributor guidelines
- Example use cases
- Launch post
- Partner outreach pack

Success criteria:
- External contributors understand how to add agents, workflows, and templates.

## Good / Better / Best Strategy

### Good
Documentation-first open-source repo.

### Better
Documentation + Notion + Supabase schema.

### Best
Documentation + Notion + Supabase + web dashboard + live demo workflows.
