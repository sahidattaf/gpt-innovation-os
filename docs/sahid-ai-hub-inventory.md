## Purpose

This inventory is the source of truth for the Sahid AI Hub. The Hub is the front door to AI projects, live apps, GPTs, GitHub repositories, Vercel deployments, Notion command centers, Claude Design systems, and Spark prototypes.

## Visibility rules

- Public cards may show public websites, public GitHub repos, public demos, and public-safe project descriptions.
- Private cards must not expose private Notion pages, GPT builder edit links, private repos, private application packages, private resumes, certificate IDs, or client data.
- Missing links should use `TBD` until verified.
- Builder/editor links should stay in Notion, not on the public Hub.
