# Claude Code Skills

This directory contains Claude Code skills for operators of GPT Innovation OS.

## Available Skills

| Skill | Description |
|-------|-------------|
| (coming soon) | Skills will be added as the platform grows |

## Adding a Skill

1. Create a directory: `skills/<skill-name>/`
2. Add `SKILL.md` with instructions
3. Add any supporting scripts

## Usage

Skills are invoked via Claude Code's `/` command interface.
